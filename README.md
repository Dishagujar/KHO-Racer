# 🏎️ KHO-Racer

A lightweight, mobile-first arcade racing game built with vanilla JavaScript, HTML5, and Tailwind CSS. 

**🌟 Features**
* **KHO-Engine:** Custom physics for smooth acceleration and braking on mobile devices.
* **Adaptive Controls:** Responsive "GamePad" style UI optimized for touch screens.
* **Chameleon Road:** Dynamic environment system where the road and world colors transition based on your score.
* **High-Speed Challenge:** Enemy cars spawn and scale speed based on player performance.

**🚀 How to Play**
1. **Tap to Start** on the splash screen.
2. Use the **Left/Right Arrows** to steer.
3. Hold **GAS** to accelerate and **BRAKE** to slow down.
4. Don't hit the blue cars!
